# Recovery Guide

This page is kept as a compatibility redirect for older recovery links.

For normal board activation, firmware update, and recovery steps, use the current
[Setup & Flash](04_setup_and_flash.md) guide. The Windows app and ST-Link flow there
are the supported paths for existing boards.

If a board is blocked by anti-rollback, UID binding, or a failed provisioning write,
contact support with:

- the license key,
- board UID, if available,
- the app log,
- the exact firmware version selected in the app.
